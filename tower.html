Sovereignty Pillars

Data Flow Architecture

Every Copilot request — completions, chat, agent actions — passes through GitHub's API proxy for intent detection and telemetry collection. This hop occurs on GitHub infrastructure regardless of your data residency or BYOK configuration. The proxy then forwards to either GitHub-managed model providers (standard path) or your own endpoint (BYOK path). Only VS Code user-level BYOK bypasses the proxy entirely.
